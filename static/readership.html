Userguide to /librarystats/v1/readership endpoint

Paths

Path: "/{language}", Query: ""

Presence: mandatory

Reponse format: JSON

Response data structure: 

        { 
            "country": string 
            "isocode": string 
            "books": uint32 
            "authors": uint32 
            "readership": uint32 
        }

  • country: The official name of the country
  • isocode: The 2-letter ISO country-code, corresponding to the one provided in the request
  • books: The amount of books available for the queried language
  • authors: The amount of unique authors
  • readerhip: The population of the country

This path takes the values of 2-letter ISO639 Set 1 language codes, i.e: "no" for Norway, "fi" for Finnland or "hu" for Hungary. The path takes a single value argument and will return a list of all countries where the language of the given country-code is spoken in the given format and data stucture.

Path: "/{language}/", Query: "?limit"

Presence: optional

Reponse format: JSON

Response data structure: 

        { 
            "country": string 
            "isocode": string 
            "books": uint32 
            "authors": uint32 
            "readership": uint32 
        }

The query takes the value of a constant integer which will be the maximum number of objects in the list returned from the request.
