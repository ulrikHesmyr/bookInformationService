Userguide to /librarystats/v1/bookcount endpoint

Paths

Path: "/", Query: "?language"

Presence: mandatory

Reponse format: JSON

Response data structure: 

        { 
            "language": string 
            "books": uint32 
            "authors": uint32 
            "fraction": float32 
        }

  • language: The language code which is the same as the value of the query
  • books: The amount of books available for the queried language
  • authors: The amount of unique authors
  • fraction: The number of books divided by the number of all books served via gutendex

This query takes 2-letter ISO language codes as values for the argument of this query, i.e: "no" for Norway, "fi" for Finnland or "hu" for Hungary. The argument of this query could either be a single value i.e: /bookcount/?language=no to retrieve a single object OR the argument to this query could be a comma-separated list of values, i.e: /bookcount/?language=no,hu which will retrieve two objects, one for each query value.

Examples

Here are some sample requests with the corresponding reponses.

Request: /bookcount/?language=hu

        [{ 
            "language": "hu",
            "books": 532, 
            "authors": 158, 
            "fraction":  0.007298369
        }]

Request: /bookcount/?language=hu,fi

[
    {
        "language": "hu",
        "books": 532,
        "authors": 158,
        "fraction": 0.007298369
    },
    {
        "language": "fi",
        "books": 2824,
        "authors": 884,
        "fraction": 0.03874172
    }
]
